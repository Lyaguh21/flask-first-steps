## üîπ 1. –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –≤—ã–±–æ—Ä–æ–º (Selection Sort)

### –ò–¥–µ—è

–ù–∞ –∫–∞–∂–¥–æ–º —à–∞–≥–µ –Ω–∞—Ö–æ–¥–∏–º **–º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç** –≤ –Ω–µ–æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–π —á–∞—Å—Ç–∏ –º–∞—Å—Å–∏–≤–∞  
–∏ –º–µ–Ω—è–µ–º –µ–≥–æ –º–µ—Å—Ç–∞–º–∏ —Å –ø–µ—Ä–≤—ã–º —ç–ª–µ–º–µ–Ω—Ç–æ–º —ç—Ç–æ–π —á–∞—Å—Ç–∏.

### –°–ª–æ–∂–Ω–æ—Å—Ç—å

- –í—Ä–µ–º—è: **O(n¬≤)**
- –ü–∞–º—è—Ç—å: **O(1)**
- –ù–µ—Å—Ç–∞–±–∏–ª—å–Ω–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞

### –ü—Ä–∏–Ω—Ü–∏–ø —Ä–∞–±–æ—Ç—ã

1. –ë–µ—Ä–µ–º –ø–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç.
2. –ò—â–µ–º –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π —Å—Ä–µ–¥–∏ –æ—Å—Ç–∞–ª—å–Ω—ã—Ö.
3. –ú–µ–Ω—è–µ–º –º–µ—Å—Ç–∞–º–∏.
4. –ü–æ–≤—Ç–æ—Ä—è–µ–º –¥–ª—è –æ—Å—Ç–∞–≤—à–µ–π—Å—è —á–∞—Å—Ç–∏ –º–∞—Å—Å–∏–≤–∞.

### –ü—Ä–∏–º–µ—Ä –Ω–∞ Python

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr

nums = [64, 25, 12, 22, 11]
print(selection_sort(nums))
```

## üîπ 2. –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –≤—Å—Ç–∞–≤–∫–∞–º–∏ (Insertion Sort)

### üìå –ò–¥–µ—è

–§–æ—Ä–º–∏—Ä—É–µ–º –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω–Ω—É—é —á–∞—Å—Ç—å –º–∞—Å—Å–∏–≤–∞, –≤—Å—Ç–∞–≤–ª—è—è –∫–∞–∂–¥—ã–π –Ω–æ–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç  
–≤ –ø—Ä–∞–≤–∏–ª—å–Ω–æ–µ –º–µ—Å—Ç–æ.

### üìä –°–ª–æ–∂–Ω–æ—Å—Ç—å

- –õ—É—á—à–∏–π —Å–ª—É—á–∞–π: **O(n)**
- –•—É–¥—à–∏–π —Å–ª—É—á–∞–π: **O(n¬≤)**
- –ü–∞–º—è—Ç—å: **O(1)**
- –°—Ç–∞–±–∏–ª—å–Ω–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞

### üß† –ü—Ä–∏–Ω—Ü–∏–ø —Ä–∞–±–æ—Ç—ã

1. –í—Ç–æ—Ä–æ–π —ç–ª–µ–º–µ–Ω—Ç —Å—Ä–∞–≤–Ω–∏–≤–∞–µ–º —Å –ø–µ—Ä–≤—ã–º.
2. –í—Å—Ç–∞–≤–ª—è–µ–º –µ–≥–æ –≤ –Ω—É–∂–Ω—É—é –ø–æ–∑–∏—Ü–∏—é.
3. –ü—Ä–æ–¥–æ–ª–∂–∞–µ–º –¥–ª—è –æ—Å—Ç–∞–ª—å–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤.

### üíª –ü—Ä–∏–º–µ—Ä –Ω–∞ Python

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1

        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1

        arr[j + 1] = key
    return arr

nums = [64, 25, 12, 22, 11]
print(insertion_sort(nums))
```

## üîπ 3. –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø—É–∑—ã—Ä—å–∫–æ–º (Bubble Sort)

### üìå –ò–¥–µ—è

–°—Ä–∞–≤–Ω–∏–≤–∞–µ–º —Å–æ—Å–µ–¥–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏ –º–µ–Ω—è–µ–º –∏—Ö –º–µ—Å—Ç–∞–º–∏,  
–µ—Å–ª–∏ –æ–Ω–∏ —Å—Ç–æ—è—Ç –≤ –Ω–µ–ø—Ä–∞–≤–∏–ª—å–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ.  
–°–∞–º—ã–π –±–æ–ª—å—à–æ–π —ç–ª–µ–º–µ–Ω—Ç "–≤—Å–ø–ª—ã–≤–∞–µ—Ç" –≤ –∫–æ–Ω–µ—Ü.

### üìä –°–ª–æ–∂–Ω–æ—Å—Ç—å

- –•—É–¥—à–∏–π —Å–ª—É—á–∞–π: **O(n¬≤)**
- –õ—É—á—à–∏–π —Å–ª—É—á–∞–π (—Å —Ñ–ª–∞–≥–æ–º): **O(n)**
- –ü–∞–º—è—Ç—å: **O(1)**
- –°—Ç–∞–±–∏–ª—å–Ω–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞

### üß† –ü—Ä–∏–Ω—Ü–∏–ø —Ä–∞–±–æ—Ç—ã

1. –ü—Ä–æ—Ö–æ–¥–∏–º –ø–æ –º–∞—Å—Å–∏–≤—É.
2. –°—Ä–∞–≤–Ω–∏–≤–∞–µ–º –ø–∞—Ä—ã —Å–æ—Å–µ–¥–Ω–∏—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤.
3. –ú–µ–Ω—è–µ–º –º–µ—Å—Ç–∞–º–∏ –ø—Ä–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏.
4. –ü–æ–≤—Ç–æ—Ä—è–µ–º –¥–æ –ø–æ–ª–Ω–æ–π —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏.

### üíª –ü—Ä–∏–º–µ—Ä –Ω–∞ Python

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr

nums = [64, 25, 12, 22, 11]
print(bubble_sort(nums))
```

## üîπ 4. –ë—ã—Å—Ç—Ä–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ (Quick Sort)

### üìå –ò–¥–µ—è

–í—ã–±–∏—Ä–∞–µ–º **–æ–ø–æ—Ä–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç (pivot)**  
–†–∞–∑–¥–µ–ª—è–µ–º –º–∞—Å—Å–∏–≤ –Ω–∞:

- —ç–ª–µ–º–µ–Ω—Ç—ã –º–µ–Ω—å—à–µ pivot
- —ç–ª–µ–º–µ–Ω—Ç—ã –±–æ–ª—å—à–µ pivot  
  –†–µ–∫—É—Ä—Å–∏–≤–Ω–æ —Å–æ—Ä—Ç–∏—Ä—É–µ–º —á–∞—Å—Ç–∏.

### üìä –°–ª–æ–∂–Ω–æ—Å—Ç—å

- –°—Ä–µ–¥–Ω—è—è: **O(n log n)**
- –•—É–¥—à–∞—è: **O(n¬≤)**
- –ü–∞–º—è—Ç—å: O(log n) (—Ä–µ–∫—É—Ä—Å–∏—è)
- –ù–µ—Å—Ç–∞–±–∏–ª—å–Ω–∞—è —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞

### üß† –ü—Ä–∏–Ω—Ü–∏–ø —Ä–∞–±–æ—Ç—ã

1. –í—ã–±–∏—Ä–∞–µ–º pivot.
2. –î–µ–ª–∏–º –º–∞—Å—Å–∏–≤.
3. –†–µ–∫—É—Ä—Å–∏–≤–Ω–æ —Å–æ—Ä—Ç–∏—Ä—É–µ–º –ø–æ–¥–º–∞—Å—Å–∏–≤—ã.
4. –û–±—ä–µ–¥–∏–Ω—è–µ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç.

### üíª –ü—Ä–∏–º–µ—Ä –Ω–∞ Python

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr

    pivot = arr[len(arr) // 2]

    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]

    return quick_sort(left) + middle + quick_sort(right)

nums = [64, 25, 12, 22, 11]
print(quick_sort(nums))
```
